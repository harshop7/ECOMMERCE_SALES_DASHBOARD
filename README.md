🛒 E-Commerce Sales Dashboard (Power BI)

📌 Project Overview
The E-Commerce Sales Dashboard is an interactive Power BI project designed to analyze sales performance, customer behavior, and product trends across different dimensions such as category, state, and payment mode.
The dashboard provides a clear view of key business metrics like revenue, profit, and quantity sold, helping stakeholders quickly identify trends, top-performing segments, and areas for improvement.

🎯 Purpose
This project aims to:
Monitor overall sales performance and profitability
Identify top-performing states, customers, and product categories
Analyze sales distribution across different payment modes
Track monthly profit trends
Enable data-driven decision-making through interactive visuals
🛠️ Tech Stack
Power BI – Dashboard creation and visualization
DAX (Data Analysis Expressions) – KPI calculations
Power Query – Data transformation and cleaning

📂 Data Source
The dataset contains simulated e-commerce sales data, including:
-Sales amount, profit, and quantity
-Customer names and transactions
-Product categories and sub-categories
-Payment modes (COD, UPI, Debit Card, etc.)
-State-wise sales distribution
-Monthly order and profit trends

⭐ Highlights

🌍 Regional Sales Analysis
Highlighted key states contributing most to revenue, enabling targeted market strategies.
